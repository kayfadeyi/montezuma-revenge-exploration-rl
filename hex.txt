------------

e.g. ssh rwg27@ogg.cs.bath.ac.uk

set up ssh key
--------------
ssh-keygen -t rsa -b 4096 -C "russell75uk@yahoo.co.uk"

validate github access
----------------------

-----

git clone git@github.com:kayfadeyi/montezuma-revenge-exploration-rl.git


create virtual env
------------------

source env/bin/activate

-------
